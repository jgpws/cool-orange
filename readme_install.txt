Installation/Theme Requirements:

A browser may download files to a Documents folder, My Documents or in a Download folder within My Documents. Check these areas for the theme file.

For a remote location (web server on the Internet):

1. Download the file cool-orange_zip.zip from Jason G. Designs

2a. Extract the zip file in Windows (right-click, select Extract All... or File menu, Extract All...)

2b. In Mac OS X (version 10.3 and up), locate the zip file and double-click it

2c. For Linux, see the eHow article titled "How to Open Zip Files in Linux", located at http://www.ehow.com/how_5004466_open-zip-files-linux.html

3. Use an FTP client to upload the inside folder (cool-orange) from your computer to your remote WordPress installation. Filezilla works well for this purpose. For information on how to connect to an ftp server using Filezilla, see http://wiki.filezilla-project.org/Using#Connecting_to_an_FTP_server. The username and password are usually the ones used from your web host

4. Upload the cool-orange folder to yourwebsitetitle.com/wp-content/themes, where yourwebsitetitle is your site's domain title

5. If you are using WordPress 3.0 or above you can install a new theme from your computer to your WordPress blog from the remote site's Admistration panel. 

*In WordPress's Admin panel, click Themes
*Choose the Install Themes tab
*From the list of links at the top, choose Upload
*Choose "Browse" or "Choose File" (this is different depending on the browser)
*From here, locate the zip file and then choose Install Now

6. Activate the theme by choosing the Activate link

For a local computer:

1. Download the file cool-orange_zip.zip from Jason G. Designs

2a. Extract file in Windows (right-click, select Extract All... or File menu, Extract All...)

2b. In Mac OS X (version 10.3 and up). Locate the zip file and double-click it

3. Move the inner cool-orange folder (by drag-and-drop or cut-and-paste) to your local site.
If you are using Apache and have WordPress installed in the htdocs folder, this would be at htdocs > wordpress > wp-content > themes

4. Activate the theme by selecting the Activate link

Cool Orange is a Sandbox child theme and is dependent on the theme to function. In order to use this theme, you must download the Sandbox theme (if you don't already have it). You can download Sandbox at http://www.plaintxt.org/themes/sandbox/ and install it to your theme directory. Installation is similar to the above instructions.


Theme Options page instructions:

How to upload a logo to replace the heading

*Upload the image from your computer to the Media Library using the Upload Logo button. Close the popup box by clicking the X in the upper right corner
*Go to the Media button at the left to access the Media Library. Look for the file you just uploaded. It should be the file at the top of the list
*Once you mouseover the image, click Edit
*In the Edit Media dialog, highlight (Ctrl a) the url in the File URL textbox
*Note the width and height of the image
*Copy (ctrl c) the url and return to this page by clicking Appearance, then Cool Orange Options
*Paste (ctrl v) the url into the Logo location box
*Paste (ctrl v) the width and height of the image into the Logo width and Logo height boxes below
